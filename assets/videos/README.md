# Video Files Directory

Place your MP4 video files in this folder. The website is configured to look for the following file names:

1. `polonya-vize-randevulari.mp4` (Polonya Vize Randevuları)
2. `karta-pobytu.mp4` (Karta Pobytu / Oturum Kartı)
3. `vize-onaylari.mp4` (Başarı Hikayeleri & Vize Onayları)
4. `schengen-vize-stratejileri.mp4` (Schengen Vizesi Stratejileri)
5. `calisma-izni.mp4` (Polonya Çalışma İzni Süreci)
6. `dosya-hazirlama.mp4` (Hukuki Dosya Hazırlama Standardı)

## Video Specifications:
- **Format:** MP4 (recommended) or WebM.
- **Resolution:** Portrait mode (e.g. 1080x1920) fits the Reels modal style perfectly.
- **Compression:** H.264/AAC for maximum browser compatibility.
